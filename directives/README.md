# Directives (Layer 1: 지침)

이 폴더에는 **SOP(Standard Operating Procedure)** 문서가 Markdown 형식으로 저장됩니다.

## 역할
- 목표, 입력/출력, 사용할 도구/스크립트, 엣지 케이스를 정의
- 자연어로 작성된 작업 지침서
- AI 에이전트가 실행 스크립트를 올바른 순서로 호출하기 위한 참고 문서

## 원칙
- Directive는 **살아있는 문서**입니다. 새로운 제약 조건, 더 나은 방법, 에러 사례를 발견하면 업데이트합니다.
- 기존 Directive를 허락 없이 덮어쓰지 않습니다.
